Run Claude Code Free — Forever

One free open-source tool routes Claude Code to free AI models, so you get the full Claude Code experience without an Anthropic bill. Setup takes about five minutes.

⏱ ~5 minutes 🌱 Beginner-friendly 🖥️ Mac, Linux & Windows

Claude Code is the best coding agent out there — but it normally bills against your Anthropic account. There's a free, open-source GitHub tool called Claude Code Router that sits between Claude Code and the model. It makes Claude Code think it's talking to Anthropic, while actually routing every request to whatever model you choose — including free ones like DeepSeek, Kimi, and GLM. Over 20,000 developers already run it this way.

🎯 The honest version: Free models are roughly 80–90% as capable as Claude on most coding tasks — excellent for everyday work, prototyping, and learning. For mission-critical or large refactors you'll still want real Claude. The beauty of the router is you can switch between free and paid models per task with one config line.

Before you start Install Claude Code

The router wraps Claude Code, so you need the Claude Code CLI installed first. If you already have it, skip ahead.

npm install -g @anthropic-ai/claude-code

Requires Node.js 18+. Run in Terminal.

npm install -g @anthropic-ai/claude-code

Requires Node.js 18+. Run in PowerShell or Command Prompt.

Step 1 Install Claude Code Router

This is the free GitHub tool that does the magic. One command installs it globally:

npm install -g @musistudio/claude-code-router

That's it — you now have a new command called ccr. The project lives at github.com/musistudio/claude-code-router if you want to read the source.

Step 2 Grab a free API key

The router needs somewhere to send requests. Both options below give you free access to top open models. Pick one — you can add both later.

Go to openrouter.ai, sign up, then open Keys → Create Key. OpenRouter hosts dozens of :free-tagged models (DeepSeek, Kimi, GLM and more) that cost literally nothing. Copy your key — it starts with sk-or-v1-.

Go to build.nvidia.com, sign up (no card needed), then API Keys → Generate Key. New accounts get 1,000 free credits on NVIDIA's hosted catalog — DeepSeek, Llama, Mistral and more. Copy your key — it starts with nvapi-.

Step 3 Drop in a config file

The router reads a single JSON file at ~/.claude-code-router/config.json. Create it and paste the matching config below, swapping in your key.

Option A — OpenRouter (free models)

{
  "Providers": [
    {
      "name": "openrouter",
      "api_base_url": "https://openrouter.ai/api/v1/chat/completions",
      "api_key": "sk-or-v1-YOUR_KEY_HERE",
      "models": [
        "deepseek/deepseek-chat-v3-0324:free",
        "moonshotai/kimi-k2:free",
        "z-ai/glm-4.5-air:free"
      ],
      "transformer": { "use": ["openrouter"] }
    }
  ],
  "Router": {
    "default": "openrouter,deepseek/deepseek-chat-v3-0324:free",
    "background": "openrouter,z-ai/glm-4.5-air:free",
    "think": "openrouter,deepseek/deepseek-chat-v3-0324:free",
    "longContext": "openrouter,moonshotai/kimi-k2:free",
    "longContextThreshold": 60000
  }
}

Option B — NVIDIA NIM (free credits)

{
  "Providers": [
    {
      "name": "nvidia",
      "api_base_url": "https://integrate.api.nvidia.com/v1/chat/completions",
      "api_key": "nvapi-YOUR_KEY_HERE",
      "models": [
        "deepseek-ai/deepseek-r1",
        "meta/llama-3.3-70b-instruct"
      ]
    }
  ],
  "Router": {
    "default": "nvidia,deepseek-ai/deepseek-r1",
    "background": "nvidia,meta/llama-3.3-70b-instruct"
  }
}
💡 What the Router section does: Claude Code uses different models for different jobs. default handles your main coding, background handles quick low-stakes tasks (cheaper/faster model), think handles deep reasoning, and longContext kicks in past longContextThreshold tokens. Point each line at whatever model you like — format is always provider,model.

Step 4 Launch Claude Code through the router

Instead of typing claude, you now start your session with:

ccr code

That boots the router and opens Claude Code wired to your free model. Everything you know about Claude Code — slash commands, file editing, the agent loop — works exactly the same. If you ever change config.json, apply it with:

ccr restart
✅ To confirm it's working, run /status inside Claude Code — you should see your routed model name, not an Anthropic model.

If you hit a wall, either wait for the reset, add a few dollars of OpenRouter credit to unlock higher limits, or route to a paid model just for that session — the router makes switching a one-line change.
