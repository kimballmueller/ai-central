Set Up Hermes Agent Desktop

The no-terminal way to run Nous Research's self-improving AI agent. Download the app, point it at a model, and you're chatting in about 10 minutes.

⏱ ~10 minutes 🌱 Beginner-friendly 🖥️ Mac, Windows & Linux

Hermes Agent is Nous Research's open-source AI agent — it remembers your projects, writes its own reusable skills, runs tasks on a schedule, and connects to tools like web search, browser automation, and image generation. Until recently it was terminal-only. As of June 2026, the official Hermes Desktop app puts the whole thing behind a clean GUI. It's free and open source under the MIT license.

🎒 What you'll need. A Mac, Windows, or Linux machine; about 10 minutes; and a way to power the agent — either an API key from a model provider (OpenAI, Anthropic, xAI, etc.) or a free Nous Portal account, which is the fastest path if you don't already have a key.

Step 1 Download the app

Grab the installer for your operating system. All downloads come straight from hermes-agent.nousresearch.com.

macOS 12 or newer. Download the .dmg installer:

https://hermes-assets.nousresearch.com/Hermes-Setup.dmg

Or just click Download for macOS on the homepage.

Windows 10 or 11. Download the .exe installer:

https://hermes-assets.nousresearch.com/Hermes-Setup.exe

Or click Download for Windows on the homepage.

Any modern distro — install from the terminal:

curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
📸 Screenshot slot — the Hermes homepage with the "Download for macOS / Windows" buttons.

Step 2 Install & open it

macOS: open the .dmg and drag Hermes into your Applications folder, then launch it.
Windows: run Hermes-Setup.exe and follow the installer, then open Hermes from the Start menu.

On first launch, Hermes installs its agent runtime in the background (into a folder it calls HERMES_HOME). Give it a moment — then you'll land on the onboarding screen.

📸 Screenshot slot — the first-run onboarding overlay (the welcome / "Choose provider" screen).
⚠️ On macOS, if you see "Hermes can't be opened because it is from an unidentified developer," right-click the app → Open → Open again. You only have to do this once.

Step 3 Connect a model

The agent needs a brain. During onboarding you'll be asked to Choose a provider — pick whichever AI service you have access to and paste in your API key.

Option A — Use your own API key

Open the Providers settings pane, choose your provider (OpenAI, Anthropic, xAI/Grok, and others are all listed), and paste your key. For xAI Grok you can sign in with a browser OAuth flow instead of a key.

Option B — Fastest path: Nous Portal

No key handy? Use Nous Portal and you'll be running in seconds. If you also installed the CLI, this is one command:

hermes setup --portal
⏭️ In a hurry to look around first? The onboarding lets you pick "Choose provider later" and skip straight into the app. You can add a key any time from Settings.
📸 Screenshot slot — the Providers settings pane with the API-key field.

Step 4 Pick your model

The model picker lives right in the chat composer — just to the left of the microphone icon. Click it to choose your model, set the reasoning effort, and toggle fast mode. Your choice is remembered per device and carries across new chats and restarts.

📸 Screenshot slot — the composer with the model picker open (left of the mic).

Step 5 Start chatting

That's it — type your first message and the agent goes to work. A few things worth exploring once you're in:

  • Left sidebar — run several agent conversations at once and switch between projects.
  • Preview pane & file browser — watch tool output stream live and browse the files the agent creates.
  • Voice I/O — talk to the agent and have it talk back.
  • Memory & skills — it learns how it solved a problem so the next similar task is faster.
📸 Screenshot slot — your first chat, with the sidebar and preview pane visible.

Prefer the terminal? (optional)

The desktop app and the CLI share the same agent core, config, and API keys — set up one and the other just works. To install the CLI:

curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
iex (irm https://hermes-agent.nousresearch.com/install.ps1)

Then reload your shell and start chatting:

hermes
🛠️ Handy commands: hermes setup (full setup wizard), hermes model (switch provider/model), hermes desktop (launch the GUI), and hermes doctor — which tells you exactly what's missing and how to fix it.
🔒 A couple of notes. Hermes Desktop is in public preview, so expect rapid updates. It can run tools and code, so review what you let it do — and for risky tasks use one of its sandboxed backends (local, Docker, SSH). Treat your API keys like passwords; never share a screenshot with a key visible.
