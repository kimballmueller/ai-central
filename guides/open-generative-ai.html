200+ AI Models, One Studio

Flux, Kling, Veo 3, Sora, Imagen 4 — every major image and video model in one open-source interface.

⏱ ~5 minutes🌱 No install required🎬 Image & Video

Higgsfield locks you into one set of models. Open Generative AI is the open-source alternative — it pulls 200+ image and video models into a single studio so you can swap between Flux, Seedance, Kling, Veo 3, Sora, or Nano Banana without leaving the interface.

Runs in your browser in under 5 minutes. You only pay for what you generate — no subscription.

Step 1 Open the studio

No install, no command line. The whole thing runs in your browser:

dev.muapi.ai/open-generative-ai

You'll see four studio modes: Image, Video, Lip Sync, and Cinema Studio. It'll ask for an API key before you can do anything — that's the next step.

Step 2 Create a muapi account

muapi is the cloud gateway that powers all the models. Sign up at muapi.ai — takes about 30 seconds. No credit card required to create the account.

Step 3 Generate your API key

In your muapi dashboard, go to Access Keys. Give your key a name (e.g. "Open Generative AI"), hit Generate Key, and copy it immediately — you only see it once. Treat it like a password.

⚠️ Copy the key right away. muapi doesn't show it again after you close the modal — you'd have to generate a new one.

Step 4 Paste the key and start creating

Back in the studio, paste your key into the API Key Required prompt and hit Connect & Start Creating. That unlocks all 200+ cloud models in one go.

Pick a studio mode, choose your model from the dropdown, write a prompt, and generate. Same workflow you know from other tools — just way more model choices.

What's in there

The model list covers every major name plus a lot of others you haven't tried yet:

Image

  • Flux — fast, high quality, great for photorealism
  • Nano Banana — stylized, cinematic output

Video

  • Seedance 2.0 — high quality video generation
  • Kling — smooth motion, strong character consistency
  • Veo 3 — Google's video model
  • Sora — OpenAI's video model
  • Runway
💡 Browse the full model list and per-model pricing at muapi.ai/playground. Expect a few cents per image and up to a couple of dollars for heavy video models like Seedance 2.0.

Optional: self-host it

If you want full ownership, the repo is MIT licensed. Clone it, run it locally, and the desktop app includes a local engine (stable-diffusion.cpp) for offline image generation with no API key at all.

github.com/Anil-matcha/Open-Generative-AI
